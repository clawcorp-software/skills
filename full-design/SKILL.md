# Full Design Power -- All Design Skills in Sequence

Shortcut for maximum design quality. Runs the full design pipeline automatically.

## When to Use
- User says "full design", "make it beautiful", "design power"
- Any UI/frontend task where quality matters
- User is tired of triggering individual design skills

## Pipeline (mandatory order)

1. **/critique** -- evaluate current state, identify issues
2. **Fix** all critique findings before proceeding
3. **/adapt** -- responsive across 480px, 768px, 1024px, 1440px
4. **/animate** -- purposeful micro-interactions
5. **/colorize** -- if too monochromatic, add strategic color
6. **/delight** -- personality, joy, memorable touches
7. **/normalize** -- ensure design system consistency
8. **/polish** -- final alignment, spacing, detail pass
9. **/frontend-preflight** -- checklist before PR

## Rules

- Present the plan to the Keymaster first
- Each step: show what changed, screenshot if possible
- Skip a step ONLY with explicit justification
- After polish: ask yourself "would I be proud to show this?" -- if no, iterate
